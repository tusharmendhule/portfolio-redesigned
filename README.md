🚀 Personal Portfolio Website
A modern, responsive Full Stack Developer Portfolio built with React and Vite to showcase skills, projects, experience, and contact information.
✨ Features
Responsive design for desktop, tablet, and mobile
Modern UI with smooth animations
Hero section with animated roles
About section
Skills section
Projects showcase
Contact section
Resume download option
Social media links (GitHub & LinkedIn)
🛠️ Tech Stack
React
Vite
JavaScript (ES6+)
HTML5
CSS3
Framer Motion
React Icons
📂 Project Structure
src/
 ├── assets/
 ├── components/
 ├── pages/
 ├── App.jsx
 └── main.jsx
⚙️ Installation
git clone <repository-url>
cd portfolio-redesigned-main
npm install
npm run dev
📦 Build for Production
npm run build
