## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/tusharmendhule/portfolio-redesigned-main.git
```

### Navigate to the Project

```bash
cd portfolio-redesigned-main
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```

---

## 📦 Production Build

Build the application:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
